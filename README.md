# nice-homepage

电影票风格个人主页，基于 HTML/CSS/JavaScript 的纯前端实现。

> 现象层：创意十足的展示页面 | 本质层：时间可视化的个人品牌 | 哲学层：编程即艺术

## 快速开始

```bash
# 1. 启动本地服务器
python3 -m http.server 8000

# 2. 访问
open http://localhost:8000/src/
```
